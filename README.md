# Glassdoor Scraper

Scrapes reviews from a Glassdoor page in an open Safari window.

## Installation

#### Install Bundler

    gem install bundler

#### Clone the Repo

    git clone git@github.com:benjaminjackson/glassdoor-scraper.git
    cd glassdoor-scraper

#### Install Dependencies

    bundle
